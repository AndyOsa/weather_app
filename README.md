
# Weather App 🌤️

A Flutter weather application built for Kevych Solutions test task.

## Features
- Splash screen with Kevych Solutions branding
- Weather based on current location
- Search weather by city name
- Current temperature, max/min, humidity
- 3-day weather forecast
- Built with Flutter for Web

## How to run
1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run -d chrome`

## Built with
- Flutter
- wttr.in API (free, no key needed)
- Geolocator package
